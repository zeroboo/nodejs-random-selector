# random-selector
A nodejs module contains utilities for randomly selecting elements in a collection.

# Features
- Selecting 
- Selecting with frequency
---
# Build
On project folder run
```npm
npm install
```
Run tests
```npm
npm test
```
---
# Usage
Install to your project:
```npm
npm install random-selector
```
